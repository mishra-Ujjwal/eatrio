# 🍽️ Eatrio – Smart Restaurant Management System

A QR-based restaurant management system designed to streamline ordering, reduce human errors, and improve operational efficiency for modern dining environments.

---

## 📌 Overview

Eatrio is a **full-stack restaurant system** that replaces traditional manual ordering with a **digital, real-time workflow**.

Customers scan a QR code, place orders directly, and the system automatically routes them to the kitchen and admin dashboards.

---

## ⚡ Key Features

* 📲 **QR-Based Ordering System**
  Customers can scan a QR code and place orders instantly without waiting for staff

* 🔄 **Automated Workflow (Customer → Kitchen → Admin)**
  Eliminates manual communication gaps and speeds up order processing

* 📊 **Admin Dashboard**
  Real-time order tracking, management, and insights

* 👨‍🍳 **Kitchen Interface**
  Streamlined view for order preparation and status updates

* ⚡ **Optimized Backend APIs**
  Reduced server load by **~20%** through efficient API handling

* ❌ **Error Reduction**
  Minimized order mistakes by removing manual entry
